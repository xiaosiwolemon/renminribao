## 存档站推荐

### 数据库（推荐，但只能看不能下）

> 这个方法不行的话，可以试试自己所在地图书馆数据库有没有，比如我这里湖北省图书馆就有，免费在网上办个读者证就能看，搜《党报党刊资料库》

以湖北省图书馆为例：
1. 访问 https://www.library.hb.cn/ ，点击登录。看到登录页上的二维码，用支付宝扫描，办理电子读书证
2. 设置密码
3. 登录
4. 在这里 https://www.library.hb.cn/resource/szzy/ 找到人民日报数据库 选择馆外访问

# 人民日报存档下载脚本

从官网 https://paper.people.com.cn/rmrb/ 下载pdf并合并

## 运行环境

需要本地安装 nodejs，高版本的最好，~~低版本能不能用我不知道~~

## 怎么用？

```bash
npm i
node ./run.js
```
