# 人民日报存档下载脚本

从官网 https://paper.people.com.cn/rmrb/ 下载pdf并合并

## 运行环境

需要本地安装 nodejs，高版本的最好，~~低版本能不能用我不知道~~

## 怎么用？

```bash
npm i
node ./run.js
```
